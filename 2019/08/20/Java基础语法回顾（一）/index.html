Java基础回顾总结：

数据类型，运算符，分支结构，循环语句部分内容的一些回顾

常量：

分类 说明
字符串常量 被双引号括起来的内容 “字符串常量”
整数常量 常见的整数 88 -78
小数常量 常见的小数 5.5 -8.8
字符常量 被单引号括起来的单个内容 ‘A’ ‘0’ ‘中’
布尔常量 true false
空常量 null

  • 分类

数据类型 关键字 所占字节 取值范围|注意事项
整数类型 byte 1 -128 ~ 127
short 2
int 4 默认的整数类型
long 8 书写的时候，需要加L
浮点类型 float 4 书写的时候，需要加F
double 8 默认的浮点类型
字符类型 char 2(可以存放一个中文字符) 0 ~ 65535
布尔类型 boolean 1 true false

运算符

1、 算术运算符：

符号 说明
+ 加法
- 减法
* 乘法
/ 除法(整数相除，结果只能得到整数！如果想要小数结果，必须有浮点类型参与运算)
% 取余
++ 自增，再原值上加1
– 自减，再原值上减1

++和–的使用

  • 单独使用：放在变量的前面和后面都是一样的。都是进行自增或自减
  • 参与运算使用
    • 如果++或–放在变量的前面，先进行自增或自减，然后再参与运算
    • 如果++或–放在变量的后面，先参与运算，然后再自增或自减

+的使用

  • 和字符参与运算

    • 使用字符对应的码表值来参与运算

  • 数据类型的提升规则

  • +号和字符串参与使用

    • 任何数据和字符串进行+号操作。最终结果都是字符串
      • 如果字符串在前面,后面的运算没有提升优先级。 做的是字符串的连续拼接
      • 如果字符串在前面，后面的运算有提升优先级。 先做运算，然后再做字符串拼接
      • 如果字符串在后面,先做前面的运算，然后再做字符串的拼接

2、 赋值运算符：

符号 说明
= 将等号右边的数据赋值给等号左边的变量
+= 将等号左右两边的数据和赋值给等号左边
-= 将等号左右两边的数据差赋值给等号左边
*= 将等号左右两边的数据乘积赋值给等号左边
/= 将等号左右两边的数据商赋值给等号左边
%= 将等号左右两边的数据余数赋值给等号左边
  • 注意
    • 扩展的赋值运算符，可以自动做强制类型转换

3、 关系运算符：

符号 说明
> 大于
>= 大于等于
< 小于
<= 小于等于
== 相等
!= 不相等

4、 逻辑运算符：

符号 说明
& 逻辑与。有false则整个结果就是false
| 逻辑或。有true则整个结果就是true
^ 逻辑异或。相同为false，不同为true
&& 短路与。（前面是false后面就不再执行了）
|| 短路或。（前面是true后面就不再执行了）
! 逻辑非。

5、 三元运算符：

  • 定义格式
  • 执行流程
    • 首先计算关系表达式的值。
    • 如果为true，表达式1就是运算结果
    • 如果为false，表达式2就是运算结果

  • java程序是由上而下顺序执行

3、 switch语句

  • 定义格式

  • 注意事项：

    • case的值不能重复

    • 对于switch语句中的表达式的数据类型，有以下几种：

      byte short char int 枚举 String

    • case没有定义顺序的要求。谁在前，谁就先匹配

    • default也可以放在任意位置。但是它永远都是最后执行的

    • 如果case中没有写break，则会出现case穿透的现象

循环语句

1、 for循环

  • 定义格式
  • 执行流程
  1. 执行初始化语句
  2. 执行条件判断语句。如果是true
  3. 执行循环体语句
  4. 执行条件控制语句
  5. 回到2->3->4 直到条件为false结束

  • 使用

    一般用于循环次数明确的情形。

2、while循环

  • 定义格式
  • 执行流程
  1. 执行初始化语句
  2. 执行
  3. 条件判断语句。如果是true
  4. 执行循环体语句
  5. 执行条件控制语句
  6. 回到2->3->4 直到条件为false结束

  • 使用

    用于循环次数不明确的时候。

3、 do-while循环

  • 定义格式
  • 执行流程
  1. 执行初始化语句
  2. 执行循环体语句
  3. 执行条件控制语句
  4. 执行条件判断语句，如果是true
  5. 2->3->4 直到条件为false结束

  • do-while循环的特点

    do-while循环会无条件的先执行一遍循环体和条件控制语句。然后再做条件判断

4、 循环嵌套

1、 break和continue关键字

  • break
    • 使用场景：可以用在循环或switch语句中
    • 作用：结束循环或结束switch语句
  • continue
    • 使用场景：只能用在循环中
    • 作用：结束本次循环，继续下一次循环

2、 循环嵌套

  • 执行特点：外层循环执行一次，内层循环执行一圈
