1. 使用环境

  • win10
  • node.js@10.15.3
  • git@ 2.16.0
  • hexo2.0.0

2. 准备工作

1. 准备自己的github账号

直接去官网注册登录即可，在此不多赘述。

2. 安装node.js、npm

3. 安装git for windows（需要有git bash）

3. 开始搭建工作

1、创建仓库

点击右上角加号New repository新建仓库

将仓库名字命名为与你的登录名相同的名字+.github.io

(xxxx.github.io)

2. 绑定域名（也可以忽略）

在控制台解析CNAME就行了。

还需要把github主页下增加一个CNAME文件，内容是你要绑定的域名。加不加www都行。

安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。

新建完成后，指定文件夹的目录如下：

init

新建一个网站。如果没有设置 folder ，Hexo 默认在目前的文件夹建立网站。

new

新建一篇文章。如果没有设置 layout 的话，默认使用 _config.yml 中的 default_layout 参数代替。如果标题包含空格的话，请使用引号括起来。

generate

生成静态文件。

选项 描述
-d, --deploy 文件生成后立即部署网站
-w, --watch 监视文件变动

该命令可以简写为

server

启动服务器。默认情况下，访问网址为： http://localhost:4000/。

选项 描述
-p, --port 重设端口
-s, --static 只使用静态文件
-l, --log 启动日记记录，使用覆盖记录格式
deploy

部署网站。

参数 描述

2. 文章插入图片

1.把主页配置文件_config.yml 里的post_asset_folder:这个选项设置为true（如果没有就不用管了）

2.在cmd中你的hexo目录下执行这样一句话npm install hexo-asset-image –save，这是下载安装一个可以上传本地图片的插件

3.安装完成后，再运行hexo new “title”来生成md博文时，/source/_posts文件夹内除了title.md文件还有一个同名的文件夹

这里面就可以添加所需要展示的图片了。
